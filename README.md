# Instalación

### Clonar repositorio
```
git clone git@github.com:AleSanchezB/treyo-clone.git
cd treyo-clone
```

### Iniciar entorno
```
python -m venv venv
source venv/bin/activate
```

### Dependencias
```
pip install fastapi uvicorn
```

### Iniciar Backend
```
uvicorn backend.main:app --reload
```

### Iniciar Frontend
```
cd frontend
npm start
```
